# Oracle Migration App

Starter design repository for an Oracle-to-Oracle migration recommendation web application.

## Current Contents

- decision engine design
- application architecture
- suggested project structure
- example rule configuration

## Core Stack

- React + TypeScript + Vite
- FastAPI + Python
- PostgreSQL
- Celery + Redis
- NGINX
- Docker Compose
- OpenTelemetry
- Oracle tool adapters for SQLcl, Data Pump, RMAN, GoldenGate, and ZDM
